Please note:

This is a preliminary Worldcon 75 programme schedule. There will be changes in programme times, rooms and participants – both additions and cancellations. There will especially be additions to Signings, Strolls with Stars and GoH programme as well as details about the Games that will be available.

The programme will be continuously updated until the very last moments of the convention, so please do check for changes before making any definitive plans! If you think there is something/someone missing from the programme, please do not panic: it is likely we are still waiting for confirmation for something and are unable to publish an item before we get it.

    This is the programme guide for Worldcon 75. It should be suitable for use on most browsers and devices. It's an instance of KonOpas, an open-source project providing conventions with easy-to-use mobile-friendly guides.
